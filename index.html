    CCom - Conditional Comments

    Pre-process your static data files!

    With CCom, you can:

    • Process a file with conditional sections to a file
    • Process a file with conditional sections to a string
    • Process a string with conditional sections to a file
    • Process a string with conditional sections to a string
    • Evaluate a single statement list to a boolean
    • ... and much more!
